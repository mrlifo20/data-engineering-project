# 🚀 Sales Data Engineering Project

## 📌 Overview
This project demonstrates an end-to-end data engineering pipeline using Python and SQL Server.

## ⚙️ What I Did
- Cleaned raw data using Python (Pandas)
- Handled missing values and duplicates
- Transformed data (dates, columns, formatting)
- Loaded data into SQL Server
- Performed SQL analysis to extract insights

## 📊 Key Insights
- Technology category generates the highest sales
- West region has the highest profit
- Top customers contribute significantly to revenue
- Some orders show negative profit (possible discount issues)

## 🛠 Tools Used
- Python (Pandas)
- Microsoft SQL Server
- Excel
